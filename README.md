# stm32f103-experiments4

Experimentin with STM32F103CXT^FR-Core 1.1 Board.
Based on STM32F103C6F6A microcontroller.

## Onboard pripherial:
 - Power LED
- UserLED - PA1

- Reset Button
- Boot0 Button
- WKUP button - PA0

 - USB Type C

## Microcontroller info
ARM Cortex M3

Flash: 32k

RAM: 10k

UART: 2

SPI: 1

I2C: 1

CAN: 1

Timers: 3

## IDE
STM CubeIDE
